# Step 2: After creating the repository on GitHub, run these commands:

# Connect local repository to GitHub remote
git remote add origin https://github.com/cyberOromo/abo-wbo-management-system.git

# Push main branch
git checkout main
git push -u origin main

# Push develop branch  
git checkout develop
git push -u origin develop

# Set develop as default branch for development

# Verify remote connection
git remote -v

# Check branch status
git branch -a
